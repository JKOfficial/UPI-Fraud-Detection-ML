# UPI Fraud Detection using Machine Learning

Fraud Detection in UPI Transactions using Machine Learning.

---

## Objective
This project aims to detect fraudulent transactions using machine learning techniques on imbalanced data.

---

## Algorithms Used
- Random Forest  
- Support Vector Machine (SVM)  
- XGBoost  

---

## Techniques Used
- SMOTE (to handle imbalanced data)

---

## Dataset
- Credit Card Fraud Dataset (Kaggle)  
- Used as simulated UPI transaction data  

---

## Evaluation Metrics
- Accuracy  
- Precision  
- Recall  
- F1 Score  

---

## Key Insight
Recall is the most important metric in fraud detection because missing fraud can lead to financial loss.
